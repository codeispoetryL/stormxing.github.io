本文在ubuntu16.04和17.04上测试通过，但不适用于17.10。

安装常用软件

说明

1、可选择性安装，以下列表没有先后顺序。
2、安装软件都是在终端中操作的，“ctrl+alt+T”或桌面右键都可打开终端。
3、你可以直接复制本篇文章中的命令直接粘贴到终端，但你需要按“ctrl+shift+v”。
4、另外 在本篇文章中，所下载的软件安装包都是默认存放在“下载”目录里的，所以你会经常看到“cd 下载”这条命令，这是因为安装软件需要进入到软件所在的目录里才能安装，你也可以直接在该目录里右键打开终端，即可省去该命令。
5、在执行安装命令的时候经常会输入密码，输入的时候密码是不显示的，输完直接回车就可以了
6、在执行安装命令的时候也经常会提示“您希望继续执行吗？[y/n]”，直接输入y后回车就可以了。

调整win10和ubuntu双系统启动顺序（可选设置）

  • 前言：装了win10和ubuntu双系统后每次开机默认进Ubuntu，如果不操作会等待10秒，对于初学者来说，win10会用的比较多一点。况且不点回车要等待10秒这个时间也是过长的。

  • 调整等待时间和系统启动顺序
    在终端中执行以下命令，即可自动打开grub目录并为“grub.cfg”文件获取root权限。
    • sudo nautilus /boot/grub/grub.cfg

  • 双击打开grub.cfg文件
    • set timeout的值就是等待时间，可改为3或者1，本文件有很多“set timeout 注意只改图片中的这一个”。
    • set default的值为为默认启动系统的顺序，从0开始计算，ubuntu系统在第一位，值就是0，windows是在第三位，将值改为2即可。

  • 改完值后，保存并运行以下命令更新grub文件
    • sudo update-grub
